Welcome to the API application factory pattern:
1. to run open within VS code
2. open a terminal and navigate to the project folder
3. run python -m venv venv
4. run venv\Scripts\activate
5. run pip install flask flask-sqlalchemy mysql-connector-python flask-marshmallow marshmallow-sqlalchemy
6. from here run app.py
7. This should provide a URL in the terminal window. Typically this is an address such as http://127.0.0.1:5000
8. CRUD routing can be accessed using postman
